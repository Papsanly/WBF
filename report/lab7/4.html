У ході лабораторної роботи ми здобули практичні навички створення модального вікна для відображення збільшених зображень галереї, використовуючи бібліотеку `basicLightbox`. Ми повторили процес створення макету галереї, забезпечуючи динамічну генерацію елементів зображень за допомогою JavaScript. Особливу увагу було приділено закріпленню навичок роботи з делегуванням подій для ефективного оброблення кліків по елементах галереї. Також ми навчилися використовувати Web Storage API для збереження та відновлення даних форми зворотного зв'язку у локальному сховищі, що дозволило покращити користувацький досвід через збереження введених даних при перезавантаженні сторінки.
